# Minions Releases

Release planning, version bumps, changelog generation, and publish targets

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-releases/sdk`) — Core library for Node.js
- **Python SDK** (`minions-releases`) — Python bindings
- **CLI Tool** (`@minions-releases/cli`) — Command-line interface
- **Web App** — Interactive playground at [releases.minions.wtf](https://releases.minions.wtf)
