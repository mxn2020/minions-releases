# Welcome to Minions Releases

We're excited to introduce **Minions Releases** — Release planning, version bumps, changelog generation, and publish targets.

## What's Included

- **TypeScript SDK**: `npm install @minions-releases/sdk`
- **Python SDK**: `pip install minions-releases`
- **CLI**: `npm install -g @minions-releases/cli`

## Getting Started

Check out the [documentation](https://releases.minions.help) for a full guide.
